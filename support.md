## Common questions

### My investment prices aren't updating
Pulse fetches prices from Yahoo Finance. Pull to refresh on the Portfolio tab forces a fresh fetch. If a ticker symbol you've added isn't recognized, confirm it matches what's on Yahoo Finance.

### My data isn't syncing between my iPhone and iPad
Pulse uses your own iCloud account for sync. Confirm:
1. Both devices are signed in to the same iCloud account (Settings → [your name at the top])
2. iCloud Drive is enabled
3. Pulse is enabled in iCloud → Apps Using iCloud

Initial sync after install can take several minutes depending on data size.

### How do I import data from another app?
Pulse supports CSV import for two data types: account ledger entries and portfolio acquisitions. See Settings → Data → Import for the expected column format.

### How do I export my data?
Settings → Data → Export produces CSV files compatible with Pulse's import format. The export is a per-type round-trip — re-importing the exported files restores the same data.

### Is my financial data private?
Yes. Pulse is local-first: your data stays on your device, with optional sync through your own iCloud account. No analytics, no third-party tracking. See the [Privacy Policy](./privacy.html) for details.

### I deleted Pulse but want to make sure my data is gone
After uninstalling, also delete the app's iCloud data: iOS Settings → [your name at the top] → iCloud → Manage Account Storage → Pulse → Delete.
